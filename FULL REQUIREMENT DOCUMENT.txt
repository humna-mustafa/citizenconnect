Website Colours
* Primary Green (#009950): Main buttons, highlights – friendly, trustworthy, modern.

1. SYSTEM OVERVIEW
A unified platform for Pakistani citizens to:
   * Access step-by-step instructions to resolve local issues

   * Connect with blood donors and recipients

   * Request emergency help and guidance

   * Provide donations to verified causes

   * Engage with the community: comment, share experiences, upvote helpful guides

Core Values: Transparency, Community Engagement, Empowerment, Social Responsibility
________________


2. CORE MODULES (TOP LEVEL)
      1. User Account System (Login, Signup, Roles)

      2. Issue Guidance System (Step-by-step instructions)

      3. Community Interaction Module (Comments, Sharing, Upvotes/Likes)

      4. Blood Donor & Recipient Module

      5. Emergency Help Module

      6. Donation Module

      7. Volunteer Guidance Module

      8. Public Transparency Dashboard (Community stats)

      9. Admin Panel (Website management)

      10. Notification System

      11. Search + Filtering System

      12. AI Module (Optional: categorize issues automatically)

________________


3. DETAILED FEATURE BREAKDOWN
A. Authentication System
A1. User Registration
         * Name, Phone number (OTP optional), Email, Password, City

         * Role Selection: Citizen, Blood Donor, Volunteer

A2. Login
            * Email/phone + password

A3. Forgot Password
               * Reset link via SMS/email

A4. User Profile
                  * Personal info edit

                  * Profile photo

                  * Badge display (Donor/Volunteer/Verified Contributor)

                  * Track contributed tips, guides, and comments
